# 麻雀Web GUI仕様

このドキュメントは、麻雀Webにおけるゲーム画面のコンポーネント仕様を記述する。

## 用語の定義

- 雀卓の最前面のプレイヤーを「自家」とする
- 自家から見て右側にいるプレイヤーを「下家」とする
- 自家から見て反対にいるプレイヤーを「対面」とする
- 自家から見て左側にいるプレイヤーを「上家」とする
- 自家の画面端から起算して最も近い角度を「正面」とする
- 以降に表現する「上部面」「下部面」「右側面」「左側面」といった表現はすべて牌の「表面」を中心とした考え方である
- 数牌はそれぞれ以下のような省略表記を行うことがある
  - 萬子：1m ～ 9m（赤五萬は0m）
  - 筒子：1p ～ 9p（赤五筒は0p）
  - 索子：1s ～ 9p（赤五索は0s）
  - 一萬、二萬、三萬の順子は 123m と表現する
  - 五筒の暗刻は 555p と表現する

## 牌コンポーネント

- 牌コンポーネントは、1つの牌をあらわすコンポーネントであり、麻雀牌の最小単位である
- 現実の牌は縦26～28mm、横19-21mm、厚さ16mm程度であることから、牌コンポーネントの縦横比も4:3を基本とする
- 牌の表面はカラーコード #e1e1d9 であり、それぞれの記号（数牌や字牌）が刻まれる
- 牌の記号（数牌や字牌）のフォントなどについては別途指定する
- 牌の裏面はカラーコード #cd8d12 であり、厚み（つまり牌の上部面、下部面、右側面、左側面）のそれぞれ1/4程度がこの色となる
- 牌コンポーネントには「牌が立っている状態の見え方」と「牌が倒れている状態の見え方」の2つのパターンが存在する
- 各プレイヤーの座席位置に対して、それぞれ「牌が立っている状態」「牌が倒れている状態」のパターンが存在する
- 立直時に捨てた牌を「立直宣言牌」という。ただし立直宣言牌が他家によって副露された場合は次順以降で副露されなかった最初の牌を立直宣言牌とする
- 牌コンポーネントの頂点は少し丸みを帯びている
- レスポンシブに対応するため、絶対サイズの指定はない
- 対面、下家、上家の手牌や捨て牌にスケール差異は設けない

### 自家の牌について

- 自家の副露していない（鳴いていない）手牌は次の見え方をする
  - 正面は牌の表面である
  - 牌の表面に加え、上部が見えた状態となる
- 自家の副露している（鳴いている）手牌は次の見え方をする
  - 正面は牌の下部である。ただし他家から取った牌は左90度回転となるため、牌の左側面が正面となる
  - 牌の下部または左側面に加え、表面が見えた状態となる
- 自家の捨て牌は次の見え方をする
  - 正面は牌の下部である。ただし立直宣言牌は左90度回転となるため、牌の左側面が正面となる
  - 牌の下部または左側面に加え、表面が見えた状態となる

### 下家の牌について

- 下家の副露していない（鳴いていない）手牌は次の見え方をする
  - 正面は牌の左側面である
  - 牌の左側面に加え、上部が見えた状態となる
  - 盤面の外側（右側）に牌の表面が向き、内側（左側）に牌の裏面が向き、下から上に並ぶ状態となる
  - 手牌は左側面と上部が 1:2 の割合で見える状態となる
- 下家の副露している（鳴いている）手牌は次の見え方をする
  - 正面は牌の左側面である。ただし他家から取った牌は左90度回転となるため、牌の上部が正面となる
  - 牌の左側面または上部に加え、表面が見えた状態となる
- 下家の捨て牌は次の見え方をする
  - 正面は牌の左側面である。ただし立直宣言牌は左90度回転となるため、牌の上部が正面となる
  - 牌の左側面または上部に加え、表面が見えた状態となる

### 対面の牌について

- 対面の副露している牌、および捨て牌は自家の上下180度回転で表示される
- 対面の副露していない（鳴いていない）手牌は次の見え方をする
  - 正面は牌の裏面である
  - 牌の裏面に加え、上部が見えた状態となる
- 対面の副露している（鳴いている）手牌は次の見え方をする
  - 正面は牌の上部である。ただし他家から取った牌は左90度回転となるため、牌の右側面が正面となる
  - 牌の上部または右側面に加え、表面が見えた状態となる
- 対面の捨て牌は次の見え方をする
  - 正面は牌の上部である。ただし立直宣言牌は左90度回転となるため、牌の右側面が正面となる
  - 牌の上部または右側面に加え、表面が見えた状態となる

### 上家の牌について

- 上家の副露している牌、および捨て牌は下家の左右180度回転で表示される
- 上家の副露していない（鳴いていない）手牌は次の見え方をする
  - 正面は牌の右側面である
  - 牌の右側面に加え、上部が見えた状態となる
  - 盤面の外側（左側）に牌の表面が向き、内側（右側）に牌の裏面が向き、下から上に並ぶ状態となる
  - 手牌は右側面と上部が 1:2 の割合で見える状態となる
- 上家の副露している（鳴いている）手牌は次の見え方をする
  - 正面は牌の右側面である。ただし他家から取った牌は左90度回転となるため、牌の下部が正面となる
  - 牌の右側面または下部に加え、表面が見えた状態となる
- 上家の捨て牌は次の見え方をする
  - 正面は牌の右側面である。ただし立直宣言牌は左90度回転となるため、牌の下部が正面となる
  - 牌の右側面または下部に加え、表面が見えた状態となる

## 手牌コンポーネント

- 手牌コンポーネントは、1人のプレイヤーの手牌を表現するコンポーネントであり、牌コンポーネントの集まりで表現される
- 基本的な牌の見え方は「牌コンポーネント」に記載の内容にしたがう
- 1人のプレイヤーの対局中の手牌は13枚であり、ツモ牌を含めると14枚から1枚を捨て、13枚になることを繰り返す
- すべての手牌は面前の状態でスタートするため、「牌が立っている状態の見え方」となり、自家以外の表面は見えない
- ポン、チー、またはカンといった動作により副露を行うと、その牌はプレイヤー向かって右端に寄り、「牌が倒れている状態の見え方」となる
- ポン、チー、カンでは、取ったプレイヤーの牌を、取ったプレイヤー側に横倒しにして配置する
  - 8pを対面からポンした場合、副露の並びは 888p となり、真ん中の8pが横倒しとなる
  - 5pを下家から明カンした場合、副露の並びは 5p5p5p5p となり、一番右の 5p が横倒しとなる
  - 79sを持っている状態で8sを上家からチーした場合、副露の並びは 879s となり、左側の8sが横倒しとなる
  - 5pを上家からポンした状態 (555p、左が横倒し) で5pを加カンした場合、横倒しにされた上部に5pが横倒しで追加される
  - 9pを暗カンした場合、副露の並びは 9999p となるが、左から順に「裏面、表面、表面、裏面」で表示される
- 横倒しは常にプレイヤーから向かって左90度回転となる
- 副露した牌を並べる領域を用いて起家表示も行う
  - 起家表示は東一局の親番となったプレイヤーの副露領域の右端に配置される
  - 起家表示は背景を #f47f0c 、文字色を #222222 とする
  - 起家表示の文字は、東場の間は「東」を表示し、南場の間は「南」を表示する
  - 起家表示があるとき、副露した牌はその左側に並べていく
  - 起家表示も各プレイヤーの方向に向けて適切に回転する
- 手牌の牌同士の間隔は0pxとし密接する
- 手牌の13枚目（自家の手牌でいうと右端の牌）のツモ牌の間隔は10pxとする
- 自家の手牌のサイズは盤面（正方形）の横幅に手牌＋ツモ牌＋副露が収まるよう自動計算して算出する

## 捨て牌（河）コンポーネント

- 捨て牌（河）コンポーネントは、プレイヤーが捨てた牌を並べるコンポーネントであり、牌コンポーネントの集まりで表現される
- 捨て牌（河）コンポーネントは、後述する情報表示コンポーネントの各辺を縁として、左端から順に牌を表面で並べていく
  - 自家の捨て牌は情報表示コンポーネントの下辺を縁としてその下部に並べていく（正面から見て左側から右側へ）
  - 下家の捨て牌は情報表示コンポーネントの右辺を縁として、その右部に並べていく（正面から見て下部から上部へ）
  - 対面の捨て牌は情報表示コンポーネントの上辺を縁として、その上部に並べていく（正面から見て右側から左側へ）
  - 上家の捨て牌は情報表示コンポーネントの左辺を縁として、その左部に並べていく（正面から見て上部から下部へ）
- 基本的な牌の見え方は「牌コンポーネント」に記載の内容にしたがう
- 捨て牌は1行を6枚とし、7枚目以降はその次の行に並べていく
- 捨て牌は最大で4行まで並べることを想定する
- 前述のとおり、立直宣言牌は各プレイヤーから見て左90度回転した状態で捨て牌に並べる

## 情報表示コンポーネント

- 情報表示コンポーネントは、対局中の各種情報を表示するコンポーネントである
- 情報表示コンポーネントは、画面の中央部に正方形で表示される
- 情報表示コンポーネントは2.5Dで表示する必要はなく、HTMLオーバーレイで表示する
- 情報表示コンポーネントは盤面（正方形）の上下左右中央に適切なサイズでオーバーレイ表示する
- 情報表示コンポーネントの1辺の長さは、牌コンポーネントの短辺（横）の長さの6倍に等しい
- 以下の情報を過不足なく表示する
  - 局数（東N局、南N局）
  - 本場（N本場）
  - リーチ棒の供託点数（N,000点）
  - 残り牌山数（N枚）
  - 各プレイヤーの点数
  - 現在の親番
  - ドラ表示牌
- 各プレイヤーの点数は情報表示コンポーネントの各辺の内側にそれぞれのプレイヤーの向きで表示する
- 現在の親番表示は各プレイヤーの点数の領域を明るく表示することで表現する
- 裏ドラは立直して和了った場合のみ局結果画面に表示する

## 牌と盤面のパースについて

- 手牌、副露された牌、捨て牌はパースで投影する
- 盤面全体としては天井からやや自家に寄るかたちで、対面の手牌は牌の裏面と上部が 1:1 の比率で見える
- 河の捨て牌は以下のように見える
  - 自家の捨て牌：牌の下部が表面の高さの20％程度見える
  - 下家の捨て牌：牌の左側面が表面の幅の20％程度見える
  - 対面の捨て牌：牌の上部が表面の高さの20％程度見える
  - 上家の捨て牌：牌の右側面が表面の幅の20％程度見える
- ただし自家の手牌については盤面と異なるパースとなり、他の牌よりやや大きめで、表面の高さに対して奥行きがその30％程度として上部が見える
- 自家の副露した牌は盤面の他の牌と同じパースとなる

## キャンパスサイズについて

- 画面全体は 16:9 を想定する（典型的には 1280x720px）
- 一般的に麻雀の盤面は正方形となるため、例えば1280x720pxの画面の場合、描画される領域は720x720pxの範囲となる
- 左右中央揃えで、余白の背景色は盤面と同じ色とする
- それぞれの手牌が正方形の辺の内側に接する
  - 自家の手牌：正方形の下辺から20px程度の余白を取る
  - 下家の手牌：正方形の右辺
  - 対面の手牌：正方形の上辺
  - 上家の手牌：正方形の左辺
- 盤面の背景色は単色 `bg-emerald-900` 相当とし、盤面自体に奥行きは持たせない

## インタラクションについて

- 牌の選択、打牌は「1回クリックで選択、2回クリックで打牌」とする
- 選択された牌は少し浮かせる
- アクションボタンは従来どおり自家の副露していない手牌の上部右側に寄せて表示する
